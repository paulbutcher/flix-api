def copy[](src: { src = String }, dst: String): Result[Unit, String] & false

Copies a file or directory from path src to path dst.

Returns Ok(()) if src was copied, and dst did not already exist. Returns Err(msg) if src was not copied because:

  • dst already exists, or
  • dst is a subpath of src, or
  • an I/O error occurred.

  • dst is not a directory, or
  • an I/O error occurred.
