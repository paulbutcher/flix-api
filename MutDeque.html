MutDeque

Definitions

def foldLeft[b, a, ef, ef, r](f: b -> (a -> b & ef \ ef'), s: b, d: MutDeque[a, r]): b & ef and r

Applies f to a start value s and all elements in d going from left to right.

That is, the result is of the form: f(...f(f(s, x1), x2)..., xn).

def foldMap[a, ef, ef, b, r](f: a -> b & ef \ ef', d: MutDeque[a, r]): b & ef and r with Monoid[b]

Returns the result of mapping each element and combining the results.

def foldRight[a, b, ef, ef, r](f: a -> (b -> b & ef \ ef'), s: b, d: MutDeque[a, r]): b & ef and r

Applies f to a start value s and all elements in d going from right to left.

That is, the result is of the form: f(x1, ...f(xn-1, f(xn, s))...).

def foldRightWithCont[a, ef, ef, r, b](f: a -> ((Unit -> b & ef and r \ ef') -> b & ef and r \ ef'), s: b, d: MutDeque[a, r]): b & ef and r

Applies f to a start value z and all elements in d going from right to left.

That is, the result is of the form: f(x1, ...f(xn-1, f(xn, z))...). A foldRightWithCont allows early termination by not calling the continuation.

def forEach[a, ef, ef, r](f: a -> Unit & ef \ ef', d: MutDeque[a, r]): Unit & ef and r

Apply the effectful function f to all the elements in the MutDeque d.

def isEmpty[a, r](d: MutDeque[a, r]): Bool & r

Returns true if d is empty.

def iterator[a, r](d: MutDeque[a, r]): Iterator[a, r] & r

Returns an iterator over d.

Modifying d while using an iterator has undefined behavior and is dangerous.

def join[a, r](sep: String, d: MutDeque[a, r]): String & r with ToString[a]

Returns the concatenation of the string representation of each element in d with sep inserted between each element.

def joinWith[a, ef, ef, r](f: a -> String & ef \ ef', sep: String, d: MutDeque[a, r]): String & ef and r

Returns the concatenation of the string representation of each element in d according to f with sep inserted between each element.

def new[r, a](r: Region[r]): MutDeque[a, r] & r

Returns an empty MutDeque.

def peekBack[a, r](d: MutDeque[a, r]): Option[a] & r

Optionally returns the back element. Does not remove it.

def peekFront[a, r](d: MutDeque[a, r]): Option[a] & r

Optionally returns the front element. Does not remove it.

def popBack[a, r](d: MutDeque[a, r]): Option[a] & r

Returns Some(x) where x is the element at the back. Returns None if d is empty.

def popFront[a, r](d: MutDeque[a, r]): Option[a] & r

Returns Some(x) where x is the element at the front. Returns None if d is empty.

def product[r](d: MutDeque[Int32, r]): Int32 & r

Returns the product of all elements in the deque d.

def productWith[a, ef, ef, r](f: a -> Int32 & ef \ ef', d: MutDeque[a, r]): Int32 & ef and r

Returns the product of all elements in the deque d according to the function f.

def pushBack[a, r](x: a, d: MutDeque[a, r]): Unit & r

Pushes x to the back of d.

def pushFront[a, r](x: a, d: MutDeque[a, r]): Unit & r

Pushes x to the front of d.

def sameElements[t, r1, r2](d1: MutDeque[t, r1], d2: MutDeque[t, r2]): Bool & r1 and r2 with Eq[t]

Returns true if MutDeques a and b have the same elements in the same order, i.e. are structurally equal.

def shuffle[r1, a, r2](r1: Region[r1], rnd: Random, d: MutDeque[a, r2]): MutDeque[a, r1] & false

Shuffles a copy of d using the Fisher–Yates shuffle.

def size[a, r](d: MutDeque[a, r]): Int32 & r

Returns the number of elements in d.

def sum[r](d: MutDeque[Int32, r]): Int32 & r

Returns the sum of all elements in the deque d.

def sumWith[a, ef, ef, r](f: a -> Int32 & ef \ ef', d: MutDeque[a, r]): Int32 & ef and r

Returns the sum of all elements in the deque d according to the function f.

def toArray[r1, a, r2](r1: Region[r1], d: MutDeque[a, r2]): Array[a, r1] & r2 and r1

Returns d as an array.

def toList[a, r](d: MutDeque[a, r]): List[a] & r

Returns d as a List.
