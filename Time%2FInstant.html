Time/Instant

Definitions

@Experimentaldef after[](x: Instant, y: Instant): Bool & true

Returns true if x is after y.

@Experimentaldef before[](x: Instant, y: Instant): Bool & true

Returns true if x is before y.

@Experimentaldef minus[](x: Instant, d: Int64): Instant & true

Returns the given instant x minus d number of nano seconds.

@Experimentaldef now[](_unit: Unit): Instant & false

Returns the current time from the system clock.

@Experimentaldef plus[](x: Instant, d: Int64): Instant & true

Returns the given instant x plus d number of nano seconds.
