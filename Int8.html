Int8

Definitions

def abs[](x: Int8): Int8 & true

Returns the absolute value of x. If the absolute value exceeds maxValue(), -1 is returned.

def bitCount[](x: Int8): Int32 & true

Returns the number of one-bits in the two's complement binary representation of x.

def clearBit[](pos: { pos = Int32 }, x: Int8): Int8 & true

Returns x with the bit at position pos cleared (to 0). Considers the 5 rightmost bits of pos (pos mod 32). The bits of x have positions: 0 (rightmost bit) - 7 (leftmost bit)

def compare[](x: Int8, y: Int8): Int32 & true

Returns 1 if x > y, -1 if x < y, and 0 if x = y. The sign of x - y.

def dist[](x: Int8, y: Int8): Int8 & true

Returns the distance between x and y. If this distance exceeds maxValue(), -1 is returned.

def factorial[](x: Int8): Int32 & true

Returns the factorial of x.

def flipBit[](pos: { pos = Int32 }, x: Int8): Int8 & true

Returns x with the bit at position pos flipped. Considers the 5 rightmost bits of pos (pos mod 32). The bits of x have positions: 0 (rightmost bit) - 7 (leftmost bit)

def fromString[](s: String): Option[Int8] & true

Parse the string s as an Int8, leading or trailing whitespace is trimmed. A successful parse is wrapped with Some(x), a parse failure is indicated by None.

def getBit[](pos: { pos = Int32 }, x: Int8): Int32 & true

Returns the bit of x at pos (either 0 or 1). Considers the 5 rightmost bits of pos (pos mod 32). The bits of x have positions: 0 (rightmost bit) - 7 (leftmost bit)

def highestOneBit[](x: Int8): Int8 & true

Returns a value with at most a single one-bit, in the position of the highest-order/leftmost one-bit in x. Returns 0 if x=0.

def highestOneBitPosition[](x: Int8): Int32 & true

Returns the position of the highest-order/leftmost one-bit in x. Possible return values: 0 (rightmost bit) - 7 (leftmost bit) -1 if x = 0

def log2[](x: Int8): Int8 & true

Returns the integer binary logarithm of x. If the given value is 0 or negative, 0 is returned.

def logicalRightShift[](dist: { dist = Int32 }, x: Int8): Int8 & true

Returns the logical right shift of x by distance. Only the rightmost 5 bits of distance are considered (ie. distance rem 32). A zero is shifted into the leftmost position regardless of sign extension.

def lowestOneBit[](x: Int8): Int8 & true

Returns a value with at most a single one-bit, in the position of the highest-order/leftmost one-bit in x. Returns 0 if x=0.

def lowestOneBitPosition[](x: Int8): Int32 & true

Returns the position of the lowest-order/rightmost one-bit in x. Possible return values: 0 (rightmost bit) - 7 (leftmost bit) -1 if x = 0

def max[](x: Int8, y: Int8): Int8 & true

Returns the larger of x and y.

def maxValue[](_unit: Unit): Int8 & true

Returns the maximum number representable by an Int8.

def min[](x: Int8, y: Int8): Int8 & true

Returns the smaller of x and y.

def minValue[](_unit: Unit): Int8 & true

Returns the minimum number representable by an Int8.

def numberOfLeadingZeros[](x: Int8): Int32 & true

Returns the number of zero bits preceding the highest-order/leftmost one-bit in x. Returns 8 if x=0.

def numberOfTrailingZeros[](x: Int8): Int32 & true

Returns the number of zero bits following the lowest-order/rightmost one-bit in x. Returns 8 if x=0.

def reverse[](x: Int8): Int8 & true

Returns the value obtained by reversing the bits in the two's complement binary representation of x.

def rotateLeft[](dist: { dist = Int32 }, x: Int8): Int8 & true

Returns the the value obtained by rotating the two's complement binary representation of x left by distance bits.

def rotateRight[](dist: { dist = Int32 }, x: Int8): Int8 & true

Returns the the value obtained by rotating the two's complement binary representation of x right by distance bits.

def setBit[](pos: { pos = Int32 }, x: Int8): Int8 & true

Returns x with the bit at position pos set (to 1). Considers the 5 rightmost bits of pos (pos mod 32). The bits of x have positions: 0 (rightmost bit) - 7 (leftmost bit)

def signum[](x: Int8): Int32 & true

Returns 1 if x > 0, -1 if x < 0, and 0 if x = 0. The sign of x.

def size[](_unit: Unit): Int32 & true

Returns the number of bits used to represent an Int8.

Convert x to a BigInt.

The numeric value of x is preserved exactly.

def toFloat64[](x: Int8): Float64 & true

Convert x to a Float64.

Convert x to an Int16.

Convert x to an Int32.

The numeric value of x is preserved exactly.

Convert x to an Int64.
