Environment

Definitions

def getArgs[](_unit: Unit): List[String] & true

Returns the arguments passed to the program via the command line.

def getCurrentWorkingDirectory[](_unit: Unit): Option[String] & true

Returns the user's current working directory

def getEnv[](_unit: Unit): Map[String, String] & true

Returns an map of the current system environment.

def getFileSeparator[](_unit: Unit): String & true

Returns the file separator

def getLineSeparator[](_unit: Unit): String & true

Returns the system line separator

def getOsArch[](_unit: Unit): Option[String] & true

Returns the operating system architecture

def getOsName[](_unit: Unit): Option[String] & true

Returns the operating system name

def getOsVersion[](_unit: Unit): Option[String] & true

Returns the operating system version

def getPathSeparator[](_unit: Unit): String & true

Returns the path separator

def getProp[](name: String): Option[String] & true

Returns the system property by name.

def getTemporaryDirectory[](_unit: Unit): Option[String] & true

Returns the default temp file path

def getUserHomeDirectory[](_unit: Unit): Option[String] & true

Returns the user's home directory

def getUserName[](_unit: Unit): Option[String] & true

Returns the user's account name

def getVar[](name: String): Option[String] & true

Returns the value of the specified environment variable.

def getVirtualProcessors[](_unit: Unit): Int32 & true

Returns the number of virtual processors available to the JVM.

The returned value is never less than one.
