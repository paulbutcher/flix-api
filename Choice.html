@Experimentaldef assertAbsent[s](c: Choice[s, true, false]): Bool & true

Asserts that c is Absent. Akin to a type ascription.

Always returns true.

@Experimentaldef assertPresent[s](c: Choice[s, false, true]): Bool & true

Asserts that c is Present. Akin to a type ascription.

Always returns true.

@Experimentaldef filter[t, a, p](f: t -> Bool, c: Choice[t, a, p]): Choice[t, a or p, p] & true

Returns Present(v) if c is Present(v) and the predicate f(v) is true. Otherwise returns None.

The function f must be pure.

@Experimentaldef flatMap[s, t, a1, p1, a2, p2](f: s -> Choice[t, a2, p2], c: Choice[s, a1, p1]): Choice[t, a1 or (p1 and a2), p1 and p2] & true

Returns f(v) if c is Present(v). Otherwise returns Absent.

@Experimentaldef flatten[t, a1, p1, a2, p2](c: Choice[Choice[t, a1, p1], a2, p2]): Choice[t, (a1 and p2) or a2, p1 and p2] & true

Returns v if c is Present(v). Otherwise returns Absent.

@Experimentaldef getWithDefault[s, pres, abs](d: s, c: Choice[s, pres, abs]): s & true

Returns v if o is Present(v). Otherwise returns d.

@Experimentaldef invert[s, a, p](c: Choice[s, a, p], v: s): Choice[s, p, a] & true

Returns Absent if c is Present(_). Otherwise returns Present(v).

@Experimentaldef isAbsent[s, pres, abs](c: Choice[s, pres, abs]): Bool & true

Returns true if c is Absent.

@Experimentaldef isPresent[s, pres, abs](c: Choice[s, pres, abs]): Bool & true

Returns true if c is Present.

@Experimentaldef map[s, t, a, p](f: s -> t, c: Choice[s, a, p]): Choice[t, a, p] & true

Returns Present(f(v)) if c is Present(v). Otherwise returns Absent.

@Experimentaldef unbox[s](c: Choice[s, false, true]): s & true

Returns v where c must be Present(v).

@Experimentaldef withDefault[s, a1, p1, a2, p2](default: { default = Choice[s, a2, p2] }, c: Choice[s, a1, p1]): Choice[s, a1 and a2, p1 or (a1 and p2)] & true

Returns c if it is Present(v). Otherwise returns default.
